Question and answers:<br>
1️⃣ What is the difference between var, let, and const?<br>
Answer: var, let and const are used to declare variables in JavaScript. var is the older way, and this can be re-declared and updated. And var only follow function scope. Where let is newer and modern. let can be updated also and it follow the block scope. Lastly const is used for values that should not change, so it can not be updated or re-declare. const follow the block scope as well.

2️⃣ What is the spread operator (...)?<br>
Answer: The spread operator (...) is used to expand the values of an array or object. It helps to copy or combine arrays and objects easily.

3️⃣ What is the difference between map(), filter(), and forEach()?<br>
Answer: map() is used to create a new array by changing each element. filter() is used to create a new array with elements that match a condition. forEach() is used to loop through each element in an array, but it does not return a new array.

4️⃣ What is an arrow function?<br>
Answer: An arrow function is a short way to write a function in JavaScript. It uses the => symbol and makes the code shorter and easier to read.

5️⃣ What are template literals? <br>
Answer: Template literals are used to create strings in JavaScript using backticks(``). They allow us to perform logical operation or JavaScript code inside a string useing ${}.
